## Installation

```bash
git clone git@github.com:brysonbw/lit-toaster.com.git lit-toaster-site
```

```bash
cd lit-toaster-site
```

```bash
npm install
```

## Usage

`Running locally`

```bash
npm run dev
```

`Build`

```bash
npm run build
```

## Contributing
